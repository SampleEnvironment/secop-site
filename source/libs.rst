Frappy
~~~~~~
`Frappy <https://forge.frm2.tum.de/review/admin/repos/secop/frappy,general>`_ is a python-based library that provides a basis for constructing SECNodes and comes with a graphical client.

Octopy
~~~~~~
Octopy is SECoP in a publish/subscribe, topics based environment: an industrial IOT centered solution with an EPICS connection. It builds its SECoP infrastructure upon MQTT and provides interfaces for easy configuration like Node-red. Development is happening in the `ESS's GitLab <https://gitlab.esss.lu.se/mesi/octopy>`_.

SHALL-libraries
~~~~~~~~~~~~~~~
The SHALL-libraries enable LabView-based programming of SECNodes. You can find the main repository `here <https://github.com/SampleEnvironment/shall>`_.

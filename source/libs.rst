SHALL library
~~~~~~~~~~~~~

The `SHALL library <https://github.com/SampleEnvironment/SHALL>`_ are a C++/Qt
based library implementing SECoP.  It can be used for programming SECNodes
in C, C++ and other languages with C FFI.  In particular, a LabView binding
is provided for easy integration of SECoP into existing LabView projects.


Frappy
~~~~~~

`Frappy <https://github.com/SampleEnvironment/frappy>`_ is a Python-based
library that provides a basis for constructing SECNodes and comes with a
graphical client.


Octopy
~~~~~~

Octopy is SECoP in a publish/subscribe, topics based environment: an industrial
IOT centered solution with an EPICS connection. It builds its SECoP infrastructure
upon MQTT and provides interfaces for easy configuration like Node-red. Development
is happening in the `ESS's GitLab <https://gitlab.esss.lu.se/mesi/octopy>`_.
